# Data-Structures-concepts-Implementation
## Overview
This project is a menu-driven C++ application that demonstrates the implementation of fundamental data structures from scratch without using the Standard Template Library (STL). The project focuses on understanding the internal working of data structures and practicing core operations through an interactive console-based interface.
## Features
### Stack
* Push elements
* Pop elements
* Peek (view top element)
* Display stack contents
### Queue
* Enqueue elements
* Dequeue elements
* Display queue contents
### Linked List
* Insert nodes
* Delete nodes
* Traverse and display the list
## Concepts Covered
* Object-Oriented Programming (OOP)
* Classes and Objects
* Dynamic Memory Allocation
* Data Structure Design
* Menu-Driven Programming
* Algorithm Analysis
* Computational Thinking
## Technologies Used
* C++
* Visual Studio Code
* GCC Compiler
## Learning Outcomes
Through this project, I strengthened my understanding of how fundamental data structures operate internally, including insertion, deletion, traversal, push/pop, and enqueue/dequeue operations. The project also helped reinforce problem-solving skills, memory management concepts, and algorithmic thinking.
## How to Run
1. Clone the repository.
2. Open the project in your preferred C++ IDE or code editor.
3. Compile the source file using a C++ compiler.
4. Run the executable.
5. Use the menu options to interact with Stack, Queue, and Linked List operations.
## Project Structure
* Stack Implementation
* Queue Implementation
* Singly Linked List Implementation
* Interactive Main Menu
* Submenus for Individual Operations
